The SolaRid Machine

The SolaRid machine collects real-time data, and directs the data to your device. The machine implements high-tech management software and eradication hardware to collect accurate sample data for your fields.

Primary Machine Components

Standing Allows the unit to be held by one primary mounting pole
Motherboard
 Central CPU that allows for the machine to operate and data to be transmitted
Solar Panels Allows the unit to operate 24/7 without running extra power to the unit
Killer Panel Insect eradication panel, tracks the amount of insects killed
Light Panels One of the multiple insect attractants
Dashboard Allows for unit operation and data retrieval

SolaRid Agricultural - IIoT Function and Capabilities

External Thermistor Measures ambient temperatures
Humidity Sensors
 Measures and displays humidity levels
Cell Modem Module Stores data in an SQL database
Strike Detection Detects voltage reduction and counts numbers of insect strikes hourly
Software Controlled Adjustments Used to turn Killing Grid system ON / OFF and to adjust Voltage Levels remotely
Precipitation Detection Detects rainfall and shuts off Killing Grid
Firmware Updates Internet downloaded updates for the machine
