The SolaRid Machine

The SolaRid machine collects real-time data, and directs the data to your device. The machine implements high-tech management software and eradication hardware to collect accurate sample data for your fields.

SolaRid Agricultural - IIoT Function and Capabilities

HardwareApplication
External Thermistor Temperature SensorMeasures ambient temperatures; applied in algorithms to determine and display insects biofix events on graphs referenced to date
Humidity Sensors
Measures and displays humidity levels to determine moisture content in air to determine freezing levels and in soil required for growth.
Plug in Cell Modem module with GPS capabilityConnects SolaRid device over a cellular network to a server that uses an SQL Database to store data collected from each unit
Strike detection on High Voltage Supply / Grid SystemMeasures Voltage Reduction and counts numbers of insect strikes (hourly) on exterior printed, Killing Grid system to determine insect densities
Software controlled adjustment on High Voltage SupplyUsed to turn Killing Grid system ON / OFF and to adjust Voltage Levels remotely, based on type of insect and crop application
Current measurementDisplays and used to discern rain (versus insect strike), shorts Killing Grid System off during rain, tests 3-minute intervals, and turn back ON after rain stops
Firmware update thru Cell Modem module via WEBApplied to upgrade each SolaRid in the field operating firmware and communicate to and operate ancillary (mating disruption dispenser) products, remotely
